### Test step

- 1: Double click "miniweb.exe".
- 2: Copy the Host from console. e.g:"10.110.24.21:8000".
- 3: Paste the host to your web browser, add "/index.html" to the tail. e.g: "http://10.110.24.21:8000/index.html".
- 4: Enjoy.
